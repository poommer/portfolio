trem-mai เป็นFinal Project รายวิชาพัฒนาเว็บด้วยภาษาและเทคโนโลยีเว็บทันสมัย trem-mai คือ เป็นเว็บไซต์เกี่ยวกับการบริการเติมเงิน เติมเกม และสตรีมมิ่งต่าง ๆ โดยพัฒนาโดยใช้ภาษา PHP CSS JavaScript และ Bootstap เนื่องจากระยะเวลาที่จำกัดทำให้สโคปของงานเป็นเพียงแค่หน้าเว็บ(fond end) ยังไม่ได้ลงระบบหลังบ้าน(Back end) สำหรับ trem-mai คือเว็บไซต์แรกของผมอาจจะมีข้อผิดพลาดเล็กน้อย แต่ผมกำลังนำกลับมาพัฒนาให้เกิดข้อผิดพลาดน้อยสุดและลงส่วนระบบหลังบ้าน
